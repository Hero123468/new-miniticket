# MiniTicket

Full-stack support ticket system built with React, TypeScript, Express, and MongoDB.

## Features

- Submit tickets with name, email, category, and description
- Toggle ticket status between **Open** and **Closed**
- Data persisted using **MongoDB**
- Clean UI built with **TailwindCSS**

## Tech Stack

- React
- TypeScript
- Express
- MongoDB
- TailwindCSS
- Vite
- Git & GitHub

## Setup Instructions

```bash
git clone https://github.com/jessefalleur/new-miniticket.git
cd new-miniticket
npm install
npm run dev
